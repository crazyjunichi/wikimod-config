# 📚 欢迎使用 WikiMod

## 使用方法
安装Bepinex并将WikiMod.dll放入Plugin文件夹。
进入游戏后可以在游戏本体的设置界面进行MOD主要功能的开关。

## ✨ WikiMod 主要功能

### 📖 内置WIKI
大多数界面点击问号可打开内置wiki
-  显示常用信息
-  兼容模组（可显示模组卡牌信息）

### 🗺️ 游戏地图
在任意游戏界面按`M`键可打开地图。
探索界面显示当前区域附近的简易地图。

#### 🌍 室外地图
-  显示家园、农田、陷阱、野生动物位置
-  显示各地点地标、动植物、可探索物品
-  显示农田、陷阱详细情况，比如农田是否成熟，陷阱是否触发。
-  显示已建造道路
-  显示狩猎围栏
-  显示动物巢穴（狐狸洞、獾洞）
-  显示山洞、尸体
-  显示动物热力图。悬停在动物图标上可查看动物常去的地方，越频繁颜色越红。
-  点击地图可以直接向目的地移动，支持遇到动物自动停止。

#### 🏢 室内地图
-  显示房间多层结构
-  显示每个房间物品
-  点击房间可快速移动

### 🌱 突出显示作物状态
-  以进度条形式显示农田、菜园的水/肥/酸/土质状态
-  显示各属性的安全范围

### 🔔 提醒
-  动物即将经过提醒
-  陷阱触发提醒
-  自定义提醒

### 📊 高级角色信息面板
- 按快捷键`C`开启
- 概览：显示角色信息、游戏记录、常用状态、特质
- 装备：分部位显示装备，支持卸下与添加装备；显示装备相关状态影响
- 伤口：分部位显示伤口，显示伤口相关状态影响
- 魔法：显示当前魔法装备，当前状态奖励，相关状态影响
- 技能：分类显示技能，显示战斗技能的招式解锁进度
- 体格：展示角色物理属性状态以及相关影响
- 精神：展示角色精神状态以及相关影响
- 消化：展示角色消化系统状态以及相关影响
- 感染：展示角色各类感染状态以及相关影响
- 化学：展示角色各种化合物状态以及相关影响

### 🔢 完整的数值显示
- 悬停显示卡牌、状态、特质、操作按钮的相关数值
- 可正确显示陷阱概率、操作条件、食物评分等

### 🔍 快速查找
- 按快捷键`Alt`开启
- 可输入中文、拼音、首字母进行搜索
- 在场景中默认搜索场景物品并跳转。包括容器中、蓝图中、装备中
- 在蓝图界面搜索指定蓝图并跳转
- 在地图界面搜索指定标签、掉落物，并高亮对应地图。

### ⚙️ 万能修改器
按`键(ESC下面那个键)激活。

-  搜索游戏内的任意数据
-  获取任意物品，可定义属性、附魔。
-  修改已有卡牌的任意数据。
-  查看或修改任意状态，清除衰减。支持锁定。
-  查看或修改NPC状态，移动NPC至身边。
-  支持自由穿梭游戏时间，并保持季节同步
-  调整全局速率：蓝图研究耗时/食物腐败速度/技能衰减/动物刷新速度/农田生长速度/菜园生长速度/农田产量/菜园产量/地图场景容量/战斗难度
-  快速建造、搬家
-  可在开局后增加、删除角色特质。
-  可以强制完成任务
-  支持加速时间动画
-  支持开发者模式。开启后可以查看一些调试数据。
-  支持不同名称卡牌不叠加
-  支持所有卡牌可删除，开启后所有卡牌的删除按钮可点击。
-  支持快速进屋，开启后进屋出屋不再需要弹窗确认。
-  支持配置数显延迟时间，避免卡牌数显闪烁。
-  支持显示已生效的其他mod的版本以及相关配置
-  支持修改数显字体。
-  更多实验性功能可以进入“更多设置”查看

### ⚡ 快速添加物品
在蓝图/容器界面槽位点击`右键`，可弹出菜单快速添加可用物品
-  可显示物品属性
-  可按属性排序优先使用，如：优先使用高品质物品

### 📈 有用的信息显示
-  可显示当前季节、日期、总存活时间
-  可显示当前温度
-  可显示当前月相
-  可显示天气预报
-  可显示重量信息
-  可显示蓝图信息
-  可显示沙拉的得分
-  突出显示作物状态

### 🏷️ 更多实用功能
-  快捷背包栏
-  支持随意重命名几乎所有卡牌
-  可以在休息选项中添加睡6小时选项 (可在高级设置中调整时长)
-  按住CTRL+点击卡牌上的箭头，可以整组移动
-  支持显示卡牌属性变化箭头
-  支持优化卡牌放在地上的默认位置
-  支持禁用动物脚印/血迹
-  在玩家负重条上显示超重的临界点
